Data Science Core

The Data Science Core provides complementary, open-source tools to store, manage, process, and share the neurophysiology data generated across the U19 Team. Our strategy is to leverage existing community platforms — in particular the DANDI Archive and the NWB standard — rather than build a separate solution from scratch, and to provide the connective tissue around them.

All tools below are listed and documented under a single umbrella repository: U19-data-management-and-processing, which links each tool as a submodule. They fall into two groups: data management and data processing & analysis.

Data management

Complementary approaches to organizing, storing, and annotating data.

Dataset manager

Guided app that takes a project from metadata-template generation to NWB conversion, validation, and visualization — producing a custom conversion script per project.

Repository
Backup and processing

Acquisition→NAS backup, metadata consolidation, and remote-trigger of cluster processing (conversion, data transfer, data repository).

Repository
Forgejo data repository

A lightweight, on-site, versioned store for project metadata (per-subject and per-session JSON) hosted on the performance-center NAS, complementing DANDI during ongoing projects.

Docs & example
BrainSTEM integration

Programmatic sync of notes and metadata with the BrainSTEM electronic lab notebook via its API — push metadata in, or export records to NWB.

Repository

Data processing and analysis

Pipelines and worked examples for turning raw data into shareable, analyzable products.

AIND ephys pipeline ↔ NWB / DANDI

Integration of the Allen Institute for Neural Dynamics (AIND) Kilosort4 spike-sorting pipeline with NWB conversion and DANDI deposition, running on an HPC cluster (SLURM + Apptainer).

Repository
Ingestion scripts

Reference library of per-modality, per-language (Python/MATLAB) conversion examples and guidelines for ephys, 2-photon, widefield, and behavior data.

Repository
NWB conversion examples

Complete, published conversion→deposition walkthroughs tied to specific Dandisets — e.g. Peri-head distance PrV (001687).
